## Backend

The backend is built with Node.js and Express.

### Installation

1. Navigate to the Backend directory:
    cd Backend
    
2. Install the dependencies:
    npm install

3. Start the server:
    npm server.js

###Frontend
   ### Installation

1. Navigate to the Frontend directory:
    cd Frontend
    
2. Install the dependencies:
    npm install

    npm run dev
